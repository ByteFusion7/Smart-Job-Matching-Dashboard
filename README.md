# Smart Job Matching Dashboard

A React + Express app that matches students to jobs and shows a dynamic gauge meter for match scores.

---

## 🚀 Features
- Add students and jobs through forms
- Backend scoring based on skills, GPA, and work authorization
- Single gauge meter that updates dynamically with match score

---

## ⚙️ Setup Instructions

### 1. Clone the Repository
```bash
git clone https://github.com/yourusername/job-dashboard.git
cd job-dashboard

2. Install Dependencies
npm install

3. Run Backend
node server.js
Backend runs on http://localhost:5000

4. Run Frontend
npm start
Frontend runs on http://localhost:3000
